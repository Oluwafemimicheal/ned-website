##This project is structure and layout for a website development

##Project Folder Structure
assets -> image, icons, documents
components -> sections
components/common -> button, navbar, footer, and all other common components
components/ui -> Cards, Loaders, Heading, and all reusable components
pages -> holds our routes pages, Home, about, service, blog
layouts -> help sharing layout views
routes -> creating out routes logic
utils -> help write reusable function logic (fetch data, scrollToTop)
hooks -> creating a reusable hooks (useFetch, useScroll)

##Library Installed are:
-react-icons
-react-router-dom
-scrollreveal
-tailwindcss
-framer-motion

##Branch Rules
main → production
dev → development
